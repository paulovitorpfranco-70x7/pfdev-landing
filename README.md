## ✦ Sobre o projeto

Landing page institucional da **PF Dev**, serviço de desenvolvimento web focado em pequenos negócios do Rio de Janeiro e região.

O site foi construído com **HTML puro + CSS + JS vanilla** — sem framework, sem dependências, sem build step. Arquivo único, carregamento instantâneo.

### Diferenciais do projeto

- **Zero dependências** — HTML, CSS e JS inline em um único arquivo
- **Performance máxima** — sem bundle, sem runtime, sem overhead
- **Identidade visual própria** — gradiente azul-violeta-magenta, tipografia Bebas Neue + DM Sans
- **Mobile first** — responsivo de 375px até ultrawide
- **SEO completo** — meta tags, Open Graph, Twitter Card, canonical

## ✦ Deploy

O projeto usa deploy contínuo via Vercel. Cada push na branch `main` dispara um novo deploy automaticamente.

```bash
# Deploy de preview
npx vercel

# Deploy em produção
npx vercel --prod
```

**URLs:**
- Produção: https://pfdev-landing.vercel.app
- Domínio: pfdev.com.br *(em configuração)*
